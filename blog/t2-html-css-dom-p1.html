How would you describe HTML, CSS and the DOM?

HTML is the basic coding and filling of a webpage. All your text, links and other bits and bobs.
CSS is the styling given to the HTML. The colouring of texts, backgrounds and the size/shape of boxes.
DOM or "Domain Object Model" is the interface used by your web browser which interprets the data/code in your html and css files. It organises it lets you view specific parts of said code.

What is meant by boxifying design?

Turning everything on a webpage into many many boxes. Big boxes, little boxes, boxes in boxes in boxes in boxes. This is how you organise the placing of you page content

What is the box model?

Seen in the DOM, it displays the details of each box. How many pixels or percentage of the page it will cover including all padding and margins etc.
