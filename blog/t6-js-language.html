In the JavaScript language, what are the uses of:

Parenthesis ()
1. Used while executing a Function, eg. "functionName()"
2. Prioritizing math equations, eg. "n * (4 + 6) = x"
3. Creating a message to console.log, eg. "console.log("Hello")"

Brackets []
1. Used to contain an array of informtation, eg. "var array1:[a, b, c]"

Braces {}
1. Contains the equation or process of a function/condition, eg. "functionName() {some random process}"
2. Can contain the properties of an object, eg. "var thing1 = {colour: blue}"

Single and Double Quotes ' ', " "
Used to indentify strings, either quotes can be used provided they are used exclusively. You cannot close a single quote string with a double quote and vice versa. Often both used together when multiple quotes are needed to distinguish which parts are stringed together, eg. ' This is a quote "within a quote" '
