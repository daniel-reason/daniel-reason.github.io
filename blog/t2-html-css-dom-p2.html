What is grid based design?

Whats all the hype about responsive webpages?

In todays tech age webpages are accessed through a multitude of machines, from PC's to mobile phones. Pages need to be responsive to the different resolutions of each device in order to display information comfortably.

What is semantic structure?

This is the use of simplistic names for HTML tags that make it easy for someone to understand it's function. Such as "title", "nav" (navigation) and "img" (image).
